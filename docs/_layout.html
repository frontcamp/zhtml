Layout Container

.layout-container centers content, sets max width and paddings via variables.

<div class="layout-container">...</div>

Grid: Structural Columns

Container & Divisions

.container (flex row, no wrap). Children use .division. Grow ratios via .dv-*:

  • .dv-2-3 → grow 2; .dv-3-4 → grow 3; …
  • .reversed flips order
<div class="container">
<div class="division dv-2-3">A</div>
<div class="division">B</div>
</div>

Content Columns

.columns with .column children, gap via --column-gap-inline, padding via --column-padding. Grow ratios via .cl-*.

<div class="columns">
<div class="column cl-1-2">Left</div>
<div class="column">Right</div>
</div>

Insertions (media/figure/quote)

.insertion aligns content: .left/.start/.float-start, .right/.end/.float-end, .center/.float-none. Use .flow-stop to clear floats.

<figure class="insertion right">...</figure>
<div class="flow-stop"></div>
