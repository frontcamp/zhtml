Breakpoints (rem-based): mobile < 60rem, phones < 40rem. Tablet range: 40–60rem.

All Mobile Devices (<60rem)

  • Text align: .mobile-ta-left|center|right|justify
  • Width presets: .mobile-full-width, .mobile-1of3-width… .mobile-100pc-width
  • Visibility: .desktop-hidden, .mobile-visible/.mobile-hidden
  • Stacking: .container.mobile-stack, .columns.mobile-stack

Tablets Only (40–60rem)

  • .tablet-ta-*, width presets .tablet-* (fractions and %)
  • .tablet-visible/.tablet-hidden, .tablet-stack

Phones (<40rem)

  • .phone-ta-*, width presets .phone-*
  • .phone-visible/.phone-hidden, .phone-stack

Print

  • .print-only, .print-show, .print-hide
  • Links and <abbr> append href/title in print
