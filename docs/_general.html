ZHTML – General

Back to Index

ZHTML provides sane defaults, CSS variables, and reset-friendly base styles. Configure variables in :root to change typography, colors, container width, and spacing.

Root CSS Variables

:root {
--html-font-size: 100%;
--layout-container-max-width: 960px;
--layout-container-margin-block: 1em;
--layout-container-padding-inline: .75em;
--body-font-color: #000;
--body-font-family: Arial, sans-serif;
--body-line-height: 1.5;
--body-background: #fff;
--h1-fs: 2.00rem; --h2-fs: 1.75rem; --h3-fs: 1.50rem;
--h4-fs: 1.35rem; --h5-fs: 1.25rem; --h6-fs: 1.15rem;
--hx-lh: 1.25; --hx-mb: 1em .5em;
--link-color: #06f; --link-hover-color: #39f;
--divider-color: currentColor;
--column-gap-inline: 1em; --column-padding: .35em 1.5em;
}

Base Elements

Global box-sizing on html, neutral margins, and body defaults (font, color, background, min-height). Reset/normalize is expected to be loaded beforehand.
