Font Size Utilities

Named: .fs-biggest, .fs-bigger, .fs-big, .fs-biggy, .fs-default, .fs-regular, .fs-smally, .fs-small, .fs-smaller, .fs-smallest. Pixel variants: .fs-6px….fs-96px.

Font Weight

.fw-100….fw-900, plus .fw-lighter, .fw-bolder.

Style & Decoration

  • .fs-italic, .fs-oblique, .fs-normal
  • .td-underline, .td-overline, .td-underline-overline, .td-strike, .td-none

Transform, Spacing, Align

  • Transform: .tt-upper, .tt-lower, .tt-capitalize
  • Letter spacing: .ls-tightest….ls-widest
  • Align: .ta-left, .ta-center, .ta-right, .ta-justify
  • Line-height: .lh-1….lh-3
  • Wrapping: .ws-nowrap, .ow-break, .hyphens, .fine-wrap
