Backwards-friendly styles for old-school HTML and deprecated attributes.

HR

<hr> is styled like .divider.

Legacy Typography Tags

  • <big> sizes: default, .half, .twice, .trice
  • <small> sizes: .half, default, .twice, .trice
  • <b>, <strong> (700/800); <i>, <em>; <strike>; <u>

Deprecated Align Attribute

Respects align="left|center|right|justify" on any element.

Example

<big class="twice">Legacy BIG</big>
<p align="justify">Old attribute still works.</p>
