ZHTML – Colors Module

Back to Index

Utility classes for color, border-color, and background-color. Includes all CSS Named Colors.

Current/Inherited/Transparent

<div class="cl-current bd-current bg-current">...</div>
<div class="cl-inherit bd-inherit bg-inherit">...</div>
<div class="cl-transparent bg-transparent">...</div>

Common Border

<div class="bd-common">1px solid currentColor</div>

Custom Color via CSS Var

<div class="cl-color bd-color bg-color" style="--c:tomato">...
</div>

Named Colors

Pattern: .cl-name / .bd-name / .bg-name (e.g., .cl-aliceblue, .bg-coral, .bd-darkslategray).
