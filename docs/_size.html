Width

w-XY, minw-XY, maxw-XY.

Units

  • px: 5…1440 → w-320px
  • em: .25…90 → maxw-60em
  • %: 1,3,5…100 → minw-33pc
  • ch: 30,45,60,75,80 → w-60ch

Specials

w-0, w-auto, w-unset, w-min-content, w-max-content, w-fit-content (and minw-*/maxw-* counterparts).

Height

h-XY, minh-XY, maxh-XY with px, em, % ranges and specials: h-0, h-auto, h-unset, maxh-none.
